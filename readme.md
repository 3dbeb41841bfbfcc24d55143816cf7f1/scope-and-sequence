### RAILS

* Setting up rbEnv and Bundler

```bash
brew install rbenv
# Make sure your .bash_profile sources your .bashrc
echo 'eval "$(rbenv init -)"' >> ~/.bashrc
# Now open a new Terminal tab or window
rbenv install 2.2.4
rbenv global 2.2.4
gem update --system
gem install bundler
cd your_project_directory
bundle install
```
